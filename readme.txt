чтобы запустить проект нужно открыть терминал в папке и написать следующие строки:

python -m venv venv

Это создаст виртуальное окружение для проекта

source venv/bin/activate
pip install flask

эти строки установят фреймворк flask в виртуальное окружение

python main.py

Запустит проект по адресу 127.0.0.1:5000
